Robot Catur Inovasi

ROCA-KU

Robot pecatur pertama di Indonesia yang menggabungkan presisi mekanik dengan algoritma cerdas untuk mengalahkan pemain kelas dunia.

Status

Aktif/Operasional

Versi

v4.2.0

Panduan Pengoperasian

Ikuti prosedur standar berikut untuk keamanan dan performa maksimal.

01

Power Initialization

Sambungkan kabel power 220V dan putar saklar utama hingga indikator LED menyala.

02

Software Link

Hubungkan kabel USB ke Laptop dan jalankan aplikasi NetBeans untuk kontrol sistem.

03

Calibration

Klik tombol 'Home' pada aplikasi agar robot melakukan kalibrasi posisi awal bidak.

Langkah Penggunaan

Ikuti 6 langkah sederhana berikut untuk memulai pertandingan pertama Anda melawan ROCA-KU.

01

Power Up

Hubungkan kabel power ke sumber listrik 220V dengan stabil.

02

Aktivasi Sistem

Putar emergency switch ke kanan hingga indikator menyala hijau.

03

Koneksi Data

Sambungkan kabel USB robot ke port laptop Anda.

04

Setup Papan

Susun bidak catur pada posisi default dengan rapi.

05

Buka NetBeans

Buka proyek software pengendali melalui IDE NetBeans.

06

Klik Run

Klik tombol Run dan robot siap memulai pertandingan.
