ROCA-KU

ROCA-KU adalah inovasi robotika yang menggabungkan kecerdasan buatan (AI) dengan mekanika presisi. Robot ini mampu menganalisis langkah catur menggunakan engine Stockfish dan mengeksekusinya secara fisik menggunakan lengan robotik (robotic arm) terhadap lawan manusia secara real-time.otomatis.

Arsitektur Sistem

Memahami alur kerja data dari otak digital ke gerakan mekanik.

1

Software Logic

Java (NetBeans) mengintegrasikan Stockfish Engine untuk menghitung langkah terbaik menggunakan algoritma Search Tree.

2

Serial Protocol

Data koordinat dikonversi menjadi sinyal serial dan dikirim melalui USB ke Mikrokontroler dengan baud rate tersinkronisasi.

3

Hardware Execution

Arduino menerjemahkan koordinat menggunakan Inverse Kinematics untuk menggerakkan motor servo secara presisi.

Panduan Pengoperasian

Pilih mode panduan sesuai kebutuhan. Hardware untuk setup fisik, Software untuk operasi aplikasi.

1

Sambungkan Kabel Power

Hubungkan kabel power ke sumber listrik 220 V agar perangkat aktif.

2

Nyalakan Sistem

Putar emergency switch ke kanan sampai lampu indikator hijau menyala.

3

Hubungkan ke Laptop

Sambungkan kabel USB robot ke laptop untuk mengakses software pengendali.

4

Kalibrasi & Persiapan Papan

Pastikan papan catur sejajar dengan basis robot dan lengan berada di Home Position (titik nol) untuk menghindari tabrakan. Susun bidak tepat di tengah petak agar sensor magnetik/kamera mendeteksi posisi dengan akurat, serta pastikan area jangkauan robot bebas dari hambatan fisik.

01

Membuka NetBeans IDE

Klik ikon NetBeans pada desktop atau melalui menu Start untuk membuka Integrated Development Environment.

2

Jalankan Program

Pilih menu File > Open Project dan arahkan ke folder source code ROCA-KU. Klik tombol Run (ikon segitiga hijau) atau tekan F6 untuk memulai antarmuka. Program ROCAKU akan muncul dengan bagian-bagian utama yang sudah ditandai.

3

Konfigurasi Port USB

Pilih Port COM yang sesuai (misal: COM3) untuk menghubungkan software dengan robot.

4

Konfigurasi Port Serial

Pilih port komunikasi (COM Port) yang terdeteksi pada aplikasi agar software dapat mengirimkan instruksi gerak ke perangkat keras robot.

5

Penentuan Mode Permainan

Tentukan mode interaksi robot. Pilih mode "Semi" (Semi-Otomatis) pada menu drop-down. Mode ini sangat disarankan agar Anda dapat memantau transisi data antara langkah manual pemain dan respons robot secara lebih terkendali.

6

Penentuan Warna Bidak Robot

Atur warna bidak yang akan dimainkan oleh komputer/robot. Dalam prosedur standar ROCA-KU, robot biasanya diatur untuk memegang bidak Hitam (Black). Pastikan pilihan ini sesuai dengan posisi bidak fisik di atas papan.

7

Inisialisasi Papan (Start)

Klik tombol "Start". Sistem akan melakukan pemindaian awal untuk memastikan posisi bidak di papan fisik sesuai dengan posisi di aplikasi.

8

Input Koordinat Manual (Manual Mode)

Jika sensor papan mengalami kendala dalam mendeteksi gerak, gunakan fitur "Manual Mode". Masukkan koordinat asal (X, Y) dan koordinat tujuan (X, Y) bidak yang Anda pindahkan secara manual di aplikasi, lalu klik tombol Kirim Data agar robot dapat memproses langkah tersebut.

9

Indikator Konfirmasi Langkah

Setelah Anda menekan tombol "Kirim Data" pada Manual Mode, perhatikan indikator pada layar aplikasi. Jika data berhasil diterima oleh sistem, akan muncul tampilan notifikasi berwarna hijau dengan tulisan "Giliranmu Bermain". Ini merupakan tanda bahwa sinkronisasi antara langkah fisik dan logika software telah sukses dilakukan.

10

Pemutusan & Terminasi Program

Untuk mengakhiri sesi permainan secara aman, pertama klik tombol "Putuskan" pada panel Koneksi Arduino untuk memutus komunikasi serial antara PC dan robot. Setelah koneksi terputus, tekan Tombol Merah bergambar Ikon Tangan (Stop) untuk menghentikan seluruh eksekusi program dan mengembalikan sistem ke status idle.

Masalah Penyebab Solusi

Sensor Macet Debu pada lensa Lidar Bersihkan dengan kain microfiber
