Robot Catur

ROCA-KU

ROCA-KU adalah robot asisten cerdas yang dirancang untuk membantu navigasi pengunjung di gedung kampus. Dilengkapi dengan Voice Recognition dan pemetaan ruangan otomatis.

85% Baterai
v2.0 Versi OS

Spesifikasi Teknis

Processor

Raspberry Pi 4 / Jetson Nano

Sensors

Lidar, Ultrasonic, Camera

Actuators

DC Gearbox with Encoder

Programming

Python / C++ (ROS)

Panduan Pengoperasian

1

Sambungkan Kabel Power

Hubungkan kabel power ke sumber listrik 220 V agar perangkat aktif.

2

Nyalakan Sistem

lPutar emergency switch ke kanan sampai lampu indikator hijau menyala.

3

Hubungkan ke Laptop

Sambungkan kabel USB robot ke laptop untuk mengakses software pengendali.

4

Siapkan Papan Catur

Atur bidak pada posisi awal permainan di papan yang tersedia.

5

Buka Aplikasi

Jalankan NetBeans untuk mengakses program pengendali ROCA-KU.

6

Jalankan Program

Klik "Run" di NetBeans, sistem siap mendeteksi bidak dan bermain otomatis.

1

Membuka NetBeans IDE

Langkah pertama adalah membuka aplikasi NetBeans IDE dengan cara klik ikon NetBeans pada desktop atau melalui menu Start. NetBeans digunakan sebagai Integrated Development Environment (IDE) untuk menjalankan dan mengontrol program ROCAKU.

2

Menjalankan Project ROCAKU di NetBeans

Setelah NetBeans terbuka, pilih menu File ‚Üí Open Project, kemudian buka folder project ROCAKU (Robot Catur Kampus UDINUS). Selanjutnya, klik tombol Run (‚ñ∂) yang telah ditandai dengan garis atau panah untuk menjalankan program.

3

Tampilan Antarmuka Program ROCAKU Muncul

Setelah project dijalankan, tampilan utama program ROCAKU akan muncul di layar. Perhatikan bagian-bagian yang telah diberi tanda garis atau panah.

4

Memilih Mode Permainan

Klik menu Mode Permainan (ditandai dengan garis/panah) untuk memilih jenis permainan, seperti Player vs Robot atau Player vs Player..

5

Pengaturan Awal Permainan

Atur tingkat kesulitan robot, warna bidak (hitam/putih), serta pengaturan lain dengan mengklik opsi yang telah ditandai pada layar.

6

Memulai Permainan

Klik tombol Start / Mulai yang ditandai dengan panah untuk memulai permainan catur.

7

Melakukan Langkah Catur

Pemain melakukan langkah dengan memilih bidak catur pada papan digital sesuai giliran.

8

Respon Robot Catur

Robot catur akan memproses langkah pemain dan menjalankan langkah balasan secara otomatis.

9

Menghentikan atau Mengulang Permainan

Untuk menghentikan permainan, klik tombol Stop / Reset. Untuk mengulang, pilih menu Restart.

10

Menutup Program dan NetBeans

Setelah selesai, tutup jendela program ROCAKU, lalu keluar dari NetBeans IDE.

Penyelesaian Masalah (Troubleshooting)

Masalah Penyebab Solusi
Robot Tidak Bergerak Tegangan baterai di bawah 11V Isi daya (Charging)
