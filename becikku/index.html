Jalankan Program

Klik "Run" di NetBeans, sistem siap mendeteksi bidak dan bermain otomatis.

01

Membuka NetBeans IDE

Klik ikon NetBeans pada desktop atau melalui menu Start.

02

Jalankan Program

Pilih menu File > Open Project dan klik tombol Run.

03

Konfigurasi Port USB

Pilih Port COM yang sesuai untuk koneksi dengan robot.

04

Konfigurasi Port Serial

Pilih port komunikasi yang terdeteksi pada aplikasi.

05

Penentuan Mode Permainan

Tentukan mode interaksi robot (disarankan mode "Semi").

06

Penentuan Warna Bidak

Atur warna bidak untuk komputer/robot (biasanya bidak Hitam).

07

Inisialisasi Papan (Start)

Klik tombol "Start" untuk pemindaian awal posisi bidak.

08

Input Koordinat Manual

Gunakan fitur "Manual Mode" jika sensor mengalami kendala.

09

Indikator Konfirmasi

Perhatikan notifikasi hijau "Giliranmu Bermain" setelah kirim data.

10

Pemutusan Program

Klik "Putuskan" dan tombol Stop untuk mengakhiri sesi.
