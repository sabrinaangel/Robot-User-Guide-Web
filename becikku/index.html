Mengenal Lebih Dekat
ROCA-KU

Robot Catur Kampus Udinus (ROCA-KU) bukan sekadar mesin. Ia adalah perpaduan presisi sensor magnetik tinggi dan kecerdasan buatan yang mampu menandingi level permainan Grand Master.

High Accuracy

Sensor Magnetik 0.1mm

Auto Move

Pencapit Otomatis

Langkah Penggunaan

Ikuti 6 langkah sederhana berikut untuk memulai pertandingan pertama Anda melawan ROCA-KU.

01

Power Up

Hubungkan kabel power ke sumber listrik 220V dengan stabil.

02

Aktivasi Sistem

Putar emergency switch ke kanan hingga indikator menyala hijau.

03

Koneksi Data

Sambungkan kabel USB robot ke port laptop Anda.

04

Setup Papan

Susun bidak catur pada posisi default dengan rapi.

05

Buka NetBeans

Buka proyek software pengendali melalui IDE NetBeans.

06

Klik Run

Klik tombol Run dan robot siap memulai pertandingan.
